# Ishna Zaka - Software Engineer & AI Developer Portfolio

A premium personal portfolio showcasing AI-powered web applications, full-stack engineering, and machine learning expertise. Built with vanilla HTML5, CSS3, and JavaScript ES2024 — matching the visual quality and animation level of professional portfolios.

## ✨ Design & Animation Highlights

- **Three.js Floating Geometries**: Rotating wireframe shapes (icosahedron, octahedron, tetrahedron) in hero and footer
- **Canvas Particle System**: Physics-based particle effects with mouse interaction
- **Staggered Animations**: Word-by-word fade-up on page load with GSAP
- **Scroll-Based Animations**: Timeline cards slide in on scroll with filled dots
- **Terminal Typewriter**: Realistic typing animation with cursor blink
- **Glowing Interactions**: Skill cards and project cards with hover effects
- **Navbar Scroll Detection**: Smart hide/show behavior on scroll direction
- **100% Vanilla**: No build tools, no npm, GitHub Pages ready

## 🌐 Getting Started

1. Clone or download this repository
2. Open `index.html` in a modern browser (Chrome, Firefox, Safari, Edge)
3. All assets load via CDN with integrity checks
4. Deploy to GitHub Pages — no build step needed!

## 📱 Responsive Breakpoints

- **Desktop**: 1024px+ — Full animations, Three.js shapes
- **Tablet**: 768px–1023px — Optimized grid, touch-friendly
- **Mobile**: <768px — Hamburger menu, single column, simplified animations
