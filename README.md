# Ishna Zaka - AI/Python Developer Portfolio

A production-grade personal portfolio built with vanilla HTML, CSS, and JavaScript featuring GSAP animations and modern design principles.

## 🎨 Features

- **Modern Design System**: Custom color palette with warm accents
- **GSAP Animations**: Smooth scroll animations and transitions
- **Responsive Layout**: Mobile-first design that works on all devices
- **Dark Theme**: Professional dark mode with accent colors
- **Performance Optimized**: Single page application with zero build tools
- **GitHub Pages Compatible**: No build process required

## 📂 Sections

1. **Hero** - Introduction with terminal card and floating particles
2. **About** - Background and personal information
3. **Skills** - Technology stack with devicons
4. **Journey** - Timeline of education and projects
5. **Projects** - Portfolio of AI and full-stack projects
6. **Code Showcase** - Terminal with typing animation
7. **FYP** - Final Year Project spotlight
8. **Contact** - Get in touch section
9. **Footer** - Social links and copyright

## 🚀 Getting Started

Simply open `index.html` in your browser. All dependencies are loaded via CDN:
- Google Fonts (Playfair Display + Inter)
- GSAP & ScrollTrigger
- Devicons for skill icons

## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- GSAP 3 + ScrollTrigger
- Canvas API for particle effects
